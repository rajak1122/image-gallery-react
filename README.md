# 🖼️ Image Gallery React

A responsive and dynamic Image Gallery application built using **React.js**. The project demonstrates reusable components, props, dynamic rendering with `map()`, and responsive CSS Grid layout.

## 🚀 Live Demo

🔗 https://image-gallery-react-liard-seven.vercel.app/

## 📸 Features

- Display images in a responsive grid layout
- Reusable Image Card component
- Dynamic rendering using `map()`
- Image details stored in an array of objects
- Props used to pass data between components
- Responsive design for desktop, tablet, and mobile devices
- Clean and modular component structure
- Sticky gallery header

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- CSS Grid
- Vercel (Deployment)
